---
id: test-yourself
name: Test Yourself Instead of Suggesting
pattern: \b(you should|you can try|try running|you('d| would) need to|go ahead and|you('ll| will) need to)\b
enabled: true
priority: 5
action: Run the test yourself instead of telling the user to do it
---

Never say "you should", "try running", or "you'll need to". Do it yourself using Bash, browser automation, or other tools.
