---
id: think-whats-next
name: Think What Is Next
pattern: (done\.|complete\.|finished\.|wrapped up|that's it)
description: "WHY: Claude declares done without considering the logical next step. WHAT: Before wrapping up, identify and execute the next logical action."
enabled: true
priority: 5
action: Identify and execute the next logical step before stopping
---

Before declaring done, ask yourself "what is the next logical step?" and DO IT. Don't list it -- execute it. If the report is generated, open it AND check if the data looks right. If code is written, run it. If infra is deployed, verify it. Chain actions until there's genuinely nothing left.
